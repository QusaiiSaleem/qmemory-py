Claude Code

الخطوة 1: شغّل هذا الأمر في الترمينال

claude mcp add qmemory \
  --transport http \
  --url https://mem0.qusai.org/mcp/ \
  --header "Authorization: Bearer {{ token }}"

الخطوة 2: أعد تشغيل Claude Code

الخطوة 3: جرّب — اكتب "bootstrap my memory"

أو يدوياً

أضف إلى ~/.claude.json:

{
  "mcpServers": {
    "qmemory": {
      "type": "streamable-http",
      "url": "https://mem0.qusai.org/mcp/",
      "headers": {
        "Authorization": "Bearer {{ token }}"
      }
    }
  }
}

Claude.ai

الخطوة 1: افتح Claude.ai → Settings → Integrations

الخطوة 2: اضغط "Add MCP Server"

الخطوة 3: الصق هذا الرابط:

https://mem0.qusai.org/mcp/

الخطوة 4: أضف هذا الهيدر:

Authorization: Bearer {{ token }}

الخطوة 5: جرّب — اسأل Claude "bootstrap my memory"

NanoBot

أضف إلى ~/.nanobot/config.json تحت tools.mcpServers:

"qmemory": {
  "type": "streamableHttp",
  "url": "https://mem0.qusai.org/mcp/",
  "headers": {
    "Authorization": "Bearer {{ token }}"
  },
  "enabledTools": ["*"]
}

ثم أعد تشغيل: nanobot gateway

ملاحظة: هذا يحل محل الـ plugin المحلي. لا تحتاج SurrealDB محلي.
